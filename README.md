Lista de Contactos
==============

Lista de contactos consumiendo el API de alegra

## Para  ejecutar el proyecto
verificar que esten instalados los paquetes nesesarios

```bash

```bash
git  clone https://github.com/peterGarciag/contact-list.git

```bash
npm i &&  bower install

```bash
ionic serve  -cs
